Actors for E-Commerce System
Primary Actors:
1. Customer - The main user who browses and purchases products
2. Guest- Unregistered user who can browse but not purchase
3. Seller/Vendor - Third-party sellers who list products
4. Admin - System administrator with full access
5. Shipping Partner - External shipping service
6. Payment Gateway - External payment processing system

 Secondary Actors:
1. Inventory System - External inventory management
2. Notification Service - Email/SMS service for alerts

Actor Descriptions:
 Customer
- Type: Human
- Description: Registered user who can browse, add to cart, and purchase products
- Responsibilities: 
  - Register/Login
  - Browse products
  - Manage shopping cart
  - Place orders
  - Write reviews
 Admin
- Type: Human
- Description: System administrator with full control
- Responsibilities:
  - Manage users
  - Monitor system
  - Generate reports
  - Configure settings

Functional Requirements 
Customer-related:
●US001:
     As a customer, I want to browse products by category so I can find items I'm interested in.
●US002: 
      As a customer, I want to search for products using keywords so I can find specific items.
●US003:
    As a customer, I want to view product details, images, and reviews so I can make informed decisions.
●US004: 
      As a customer, I want to add products to my shopping cart so I can purchase multiple items together.
●US005: 
       As a customer, I want to create an account so I can save my information and track orders.
●US006:
       As a customer, I want to securely checkout and make payments so I can complete purchases.
●US007: 
    As a customer, I want to track my order status so I know when to expect delivery.
●US008: As a customer, I want to write product reviews so I can share my experience.
●US009:
     As a customer, I want to return products so I can get refunds for unsatisfactory items.
Admin-related:
●US010: 
    As an admin, I want to manage the product catalog so I can add, update, or remove products.
●US011: 
    As an admin, I want to manage user accounts so I can handle customer issues.

●US012: 
    As an admin, I want to view sales reports so I can analyze business performance.
●US013: 
    As an admin, I want to process orders so I can update status and handle exceptions.
Seller-related:
●US014: 
      As a seller, I want to manage my product listings so I can sell items on the platform.
●US015: 
      As a seller, I want to view my sales and earnings so I can track performance.
●US016: 
      As a seller, I want to manage inventory so I can update stock levels.
